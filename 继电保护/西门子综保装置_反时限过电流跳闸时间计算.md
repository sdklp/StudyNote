# IEC反时限计算公式（最常用） #
$$
t=TMS*\frac{k}{(\frac{I}{Is})^a-1}
$$
t：跳闸时间(秒)；

I：实际故障电流；

Is:设定跳闸电流，对应参数里  Threshold

k,a:曲线不同，参数不同，由以下两个参数共同决定

- Method of measurement
- Type of character.curve

TMS:时间倍率，对应参数里的Time dial

# 二、各种反时限曲线参数

| 曲线                           | k    | α    |
| ------------------------------ | ---- | ---- |
| Standard Inverse（标准反时限） | 0.14 | 0.02 |
| Very Inverse（很反时限）       | 13.5 | 1    |
| Extremely Inverse（极反时限）  | 80   | 2    |
| Long-time Inverse（长反时限）  | 120  | 1    |
